================================================================================
ENDPOINTS ADDED
================================================================================

1. GET /api/obs/audio
   - Lists all audio sources using audioManager.getAudioSources()
   - Returns: { sources: [...] }

2. GET /api/obs/audio/:inputName
   - Gets full details for a single audio source
   - Retrieves volume (dB and multiplier), mute state, and monitor type
   - Returns: { inputName, volumeDb, volumeMul, muted, monitorType }

3. PUT /api/obs/audio/:inputName/volume
   - Sets volume for an audio source
   - Body: { volumeDb: number }
   - Returns: { success, inputName, volumeDb }

4. PUT /api/obs/audio/:inputName/mute
   - Sets mute state for an audio source
   - Body: { muted: boolean }
   - Returns: { success, inputName, muted }

5. PUT /api/obs/audio/:inputName/monitor
   - Sets monitor type for an audio source
   - Body: { monitorType: string }
   - Returns: { success, inputName, monitorType }

6. GET /api/obs/audio/presets
   - Lists all audio presets (default + user presets)
   - Requires active competition
   - Returns: { presets: [...] }

7. POST /api/obs/audio/presets
   - Saves current mix as a new preset to Firebase
   - Body: { id, name, description?, sources: [{inputName, volumeDb, muted}] }
   - Requires active competition
   - Returns: { success, preset: { id, name } }

8. PUT /api/obs/audio/presets/:presetId
   - Loads and applies a preset from Firebase
   - Requires active competition
   - Returns: { success, preset: { id, name }, applied: number, errors: [...] }

9. DELETE /api/obs/audio/presets/:presetId
   - Deletes a user preset (default presets cannot be deleted)
   - Requires active competition
   - Returns: { success, deleted: presetId }
